
# Farsava - ASR Api


## install dependencies

```bash
npm install
```

## configs
```javascript
const apiConfigs = {
  baseUrl: 'https://api.amerandish.com/v1',
  actionUrl: '/speech/asr',
  authKey: '<YOUR_API_KEY>',
};

const filePath = path.resolve('<YOUR_WAV_FILE_PATH>');
```

## run

```bash
npm start
```
or
```bash
node index.js
```
